# OIBSIP Internship Tasks Repository

This repository contains the tasks completed during my internship at **Oasis Infobyte**.

## Tasks Overview

### **Task 2 BMI Calculator**
A simple and interactive Body Mass Index calculator built with HTML, CSS, and JavaScript.

**Features:**
- Input fields for height and weight  
- Instant BMI calculation  
- Category-based result (Underweight, Healthy, Overweight, Obese)  
- Clean user interface  

**Folder:**  
`Task2 BMI Calculator/`

### **Task 5 Password Generator**
A dynamic password generator that creates strong passwords based on user preferences.

**Features:**
- Adjustable password length  
- Toggle for uppercase, lowercase, numbers, and symbols  
- Random strong password generation  
- Copy-to-clipboard option  

**Folder:**  
`Task5 Password Generator/`

### **Task 6 Get Weather**
A weather application built using a real-time weather API.

**Features:**
- Search weather by city  
- Shows temperature, humidity, wind speed, and condition  
- Responsive and clean UI  
- Uses `fetch()` to call external API  

**Folder:**  
`Task6 GetWeather/`
